Design and Code “Spooky Bricks” from Scratch

Spooky Bricks is a complete, step-by-step course where you’ll build a fully playable and polished Breakout-style game using the Godot 4 game engine. Whether you're new to Godot or game development in general, this course will walk you through the entire process—from setting up your first scene to implementing complex powerups and designing a cohesive UI.

This is a hands-on course. You’ll build as you learn, applying each new concept immediately to your own version of the game. Every lesson is practical, beginner-friendly, and designed to leave you with not just knowledge—but a finished game.

What You’ll Learn

  • Build a fully functional Breakout-style game using Godot 4
  • Use Tilemaps to create reusable, flexible level layouts
  • Create dynamic interactions using PhysicsBodies
  • Design and implement modular powerups and game logic
  • Track player lives, level progression, win/loss conditions
  • Add polish with particles, sound, music, and theming
  • Build custom UI components like menus, sliders, and screens
  • Use tools like a palette plugin to streamline your visuals
  • Troubleshoot bugs and refine your game for a pro finish

Requirements

  • No prior Godot or coding experience needed
  • A computer capable of running Godot 4
  • Curiosity and willingness to learn
  • Optional: Basic comfort using software interfaces

Who This Course is For

  • New game devs looking for a structured project
  • Visual learners who prefer building over watching
  • Godot 4 beginners who want a real-world result
  • Unity/Unreal users exploring Godot’s strengths

What’s Inside the Course

  • 🎮 Start from scratch: Set up your Godot project and first scene
  • 🕹️ Make it playable: Animate the bat, launch the ball, and break bricks
  • ⚡ Add powerups: Magnetic bats, multiballs, energy bolts, and more
  • 🎨 Polish it: Sound, music, particles, and juice
  • 🏁 Wrap it up: Levels, lives, menus, and game over screens
